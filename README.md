### Install

```sh
npm install @swiftcarrot/webpacker --save-dev
yarn add @swiftcarrot/webpacker --dev
```

### Usage

```sh
webpacker serve -l http://localhost:5000
webpacker watch
webpacker build
webpacker analyze
```
